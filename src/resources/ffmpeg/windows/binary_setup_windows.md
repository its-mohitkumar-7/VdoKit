# FFmpeg Binary Setup for Windows

---

## Step 1: Download Binary
- Visit: https://www.ffmpeg.org/

### Alternative trusted builds:
- https://github.com/BtbN/FFmpeg-Builds/releases
- https://www.gyan.dev/ffmpeg/builds/

## Step 2: Extract Files
- Download the latest ZIP file
- Extract it

## Step 3: Locate Binary
- Inside extracted folder:
```
bin/ffmpeg.exe
```

## Step 4: Rename & Store
- Keep filename as:
```
ffmpeg.exe
```
- Copy it to **src/resources/ffmpeg/windows/**

---

### Notes
- FFmpeg comes with:
  - ffmpeg.exe
  - ffprobe.exe
  - ffplay.exe

- Keeping **ffmpeg.exe** file is recommended

- If links don’t work, navigate manually:
```
ffmpeg website → Downloads
```

- Official guide:
https://trac.ffmpeg.org/wiki/CompilationGuide
